# Expenova Client

Expenova is an expense tracking frontend built with React, TypeScript, and Vite.

It helps users monitor spending, categorize expenses, and keep a clean overview of their budget data.

## Features

- Track daily expenses and income
- View expense categories and totals
- Manage authentication flows with Redux and React Router
- Responsive UI built with PrimeReact components
- Easy local development and production build with Vite

## Project Structure

- `src/` - application source files
- `src/modules/` - feature modules like authentication
- `src/redux/` - Redux store and actions
- `src/routes/` - application routing and protected route handling
- `src/common/` - reusable shared components
- `src/utility/` - shared constants and helpers

## Getting Started

### Install dependencies

```bash
npm install
```

### Run in development

```bash
npm run dev
```

Open the displayed URL in your browser to preview the app.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
